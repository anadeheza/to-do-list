# to-do-list
Made with HTML, CSS and JavaScript. Using this simple but efficient to-do list with a cute, pinkish display, you can organize all your tasks! The steps are pretty straightforward, just add the task you need to get done (using the Add buton or just the enter key), with an optional personalized tag. When you have completed it, you can just cross it by clicking on top of the task or, if it does not need to be done, you can either delete that task or even all of them, but don't worry, if you press the delete all button, there will be a warning asking for confirmation, so, no, you won't lose everything by clicking it by accident.
